# Планирование новой итерации

### 1. Цель итерации
Разработать базовый модуль генерации синтетических данных с настраиваемыми параметрами распределения и встроенной валидацией выходного формата, чтобы предоставить инженерам надёжный инструмент для тестирования алгоритмов без использования реальных персональных данных.

### 2. Бэклог итерации
- Как ML-инженер, хочу задавать параметры распределения и объём выборки через конфигурационный файл, чтобы быстро генерировать данные под конкретные сценарии тестирования без правки кода. - 5 сторипойнтов
- Как ML-инженер, хочу получать синтетический датасет с сохранением логических связей между признаками, чтобы модели обучались на реалистичных зависимостях, а не на случайном шуме. - 8 сторипойнтов
- Как ML-инженер, хочу автоматически проверять сгенерированные данные на соответствие заданным типам и диапазонам, чтобы быть уверенным в корректности входных данных перед запуском пайплайна. - 5 сторипойнтов
- Как ML-инженер, хочу экспортировать результаты в стандартные форматы с фиксацией метаданных, чтобы легко интегрировать датасет в существующие тестовые окружения. - 3 сторипойнта

### 3. Список задач
**Задачи по конфигурации**
- Реализовать парсинг конфигурационного файла с параметрами генерации
- Добавить валидацию входных параметров на наличие обязательных полей
- Настроить маппинг параметров на внутренние функции генератора
- Покрыть модуль конфигурации юнит-тестами

**Задачи по генерации данных**
- Разработать базовые алгоритмы генерации категориальных и числовых признаков
- Реализовать механизм сохранения корреляций и логических связей между колонками
- Настроить балансировку целевой переменной согласно заданному распределению
- Добавить логирование процесса генерации для отладки

**Задачи по валидации**
- Написать скрипт проверки типов данных и допустимых диапазонов значений
- Реализовать проверку отсутствия нежелательных пропусков и дубликатов
- Интегрировать валидацию в основной поток генерации с прерыванием при ошибках
- Подготовить отчёт о результатах проверки в текстовом формате

**Задачи по экспорту и интеграции**
- Реализовать сохранение датасета в формате CSV с корректной кодировкой
- Добавить генерацию сопроводительного файла с метаданными и версией схемы
- Настроить установку инструмента
- Написать документацию по быстрому запуску и примерам конфигураций
